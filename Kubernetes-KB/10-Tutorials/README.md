# Tutorials

Hands-on, step-by-step walkthroughs for learning Kubernetes from scratch.

## What This Section Is For

Guided learning journeys where you build something real while understanding concepts. Each tutorial combines explanation with hands-on practice.

## When to Use This Section

- You're new to Kubernetes and want guided practice
- You want to learn by doing, not just reading
- You want a complete example from start to finish
- You have 30-60 minutes to dedicate to learning

## When to Use Other Sections Instead

- **Quick answer** → [[20-Tasks]]
- **Need to understand theory** → [[30-Concepts]]
- **Need exact syntax** → [[40-Reference]]
- **Need to diagnose a problem** → [[50-Troubleshooting]]

## How Tutorials Are Structured

Each tutorial includes:
- **Learning objectives** (what you'll understand)
- **Prerequisites** (what you need before starting)
- **Hands-on walkthrough** (step-by-step with copy-paste commands)
- **Verification** (how to know it worked)
- **Next steps** (what to explore next)
